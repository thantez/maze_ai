    Maze AI v0.5.0 MazeServer.MazeAi View Source

    this is an AI for maze game that developed with elixir it solve maze with this algorithms:

    • Breadth-first search (BFS)
    • Iterative deepening depth-first search (IDS)
    • A search (A Star) all of this algorithms will use graph search algorithm with * special frontier rules or heuristical function.
