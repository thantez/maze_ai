    MazeServer.MazeAi

    this is an AI for maze game that developed with elixir it solve maze with this algorithms

    MazeServer.MazeAi.AStar

    this module define A* rules. in this algorithm, frontier is same as BFS with a tiny difference! frontier will sort by ascending path_cost order after every push.

    MazeServer.MazeAi.BFS

    this module define BFS rules. in this algorithm, frontier is same as queue! (FIFO queue)

    MazeServer.MazeAi.IDS

    this module define IDS rules. in this algorithm, frontier is same as stack!
